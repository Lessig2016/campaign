== Description ==

"WP VI Post Duplicator" plugin for copy a post/page or custom post. You can also copy post to another custom post type. 


You can also customize setting for following features:

* Options for choose what to copy:
	- Original post/page excerpt
	- Original post/page attachments
	- Original post/page status (draft/published).
	- Original post/page date
	- All the children.
	- Taxonomies and custom fields
* Option for copy post to current post type or another listed post types in edit screen of posts.
* Options for choose post types (Post-types of your websites) for which you want to make copy of current post.
* Multi-languages support is also available.
* POT file is added for plugin translation into your native language.

== Installation ==

1. Download the "WP VI Post Duplicator" Plugin (Plugin zip file).
2. Extract it in the `/wp-content/plugins/` directory
3. Activate the plugin through the 'Plugins' menu in WordPress
4. (Optional) Customize the plugin in the "WP VI Post Duplicator" menu.

Visit our <a href="http://vivacityinfotech.net/blog/wp-vi-post-duplicator/">demo page</a> for more information.

== Frequently Asked Questions ==

= How can copy post to another post type? =

First, please enable checkbox under display settings on plugin page for copy post/page into another post-types.
your all post-types will be listed into post edit screen, please choose post-types for which you want to make copy of post and make sure saved changes into your post.
Now, you can create copy to another post type by clicking on "copy to listed post-type" button on edit screen of post or post listing page.
